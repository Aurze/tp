Detailed Description

Prototype 1 - LOG550.

Author
Guillaume Chevalier, Karl Thibault, Nicolas Arsenault
Version
0.1
Date
02 février 2017
Système d’acquisition de données en temps réel
Objectifs
• Se familiariser avec l'environnement matériel/logiciel pour le développement
− Maîtriser l’interface de Atmel Studio 6.2.
− Être capable de compiler et de télécharger un programme fourni en exemple
− Être capable de créer un nouveau projet, de le compiler et de le télécharger
• Explorer la vaste documentation disponible
• Lire, analyser et comprendre les programmes
• Développer la structure du programme d’acquisition sur les microcontrôleurs.
Les modules ASF suivants sont nécessaires pour compiler ce code :
• ADC - Anagloc to Digital Converter
• GPIO - General-Purpose Input/Output (driver)
• INTC - Interrupt Controller (driver)
• PM - Power Manager (driver)
• TC - Timer/Counter (driver)
• USART - Universal Synchronous/Asynchronous Receiver/Transmitter
• Generic board support (driver)
• System Clock Control (service)
Veuillez brancher l'EVK1100 à l'ordinateur via le port USART0.
